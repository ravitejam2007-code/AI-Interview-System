# Datasets

This directory contains standard dataset files used for training classifiers and populating the question pools.

## Datasets List

1. **`interview_questions.csv`**: Pool of standard technical questions categorized by skill (Python, React, ML, SQL, etc.).
2. **`resumes_dataset.csv`**: Text dataset of parsed resumes with labeled job categories for classifier training.
3. **`emotion_dataset.csv`**: Face expression data references or labels used during model validation.
